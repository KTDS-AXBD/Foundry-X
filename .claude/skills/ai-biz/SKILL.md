# ai-biz — AI 사업 분석 스킬셋

AX BD 2단계 발굴 프로세스에서 사용하는 AI 사업 분석 도구 11종이에요.
각 서브스킬은 독립적으로 호출할 수 있어요.

## 서브스킬 목록

| 스킬 | 용도 |
|------|------|
| `cost-model` | AI 제품 원가 구조 분석 (데이터/학습/추론/인프라 비용) |
| `feasibility-study` | AI 적용 타당성 평가 |
| `regulation-check` | AI 규제/컴플라이언스 점검 |
| `ecosystem-map` | AI 생태계 매핑 |
| `partner-scorecard` | 파트너 평가 스코어카드 |
| `pilot-design` | 파일럿 프로젝트 설계 |
| `moat-analysis` | 경쟁 해자 분석 |
| `ir-deck` | IR 자료 구조화 |
| `scale-playbook` | 스케일업 전략 플레이북 |
| `data-strategy` | 데이터 전략 수립 |
| `build-vs-buy` | Build vs Buy 의사결정 |

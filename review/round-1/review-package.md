# Foundry-X 검토 패키지 — Round 1

**날짜:** 2026-03-16
**PRD 버전:** v2 (기술 스택 확정)

---

## 사용 방법

1. 각 프롬프트 파일을 열어 하단의 GitHub 링크(https://github.com/AXBD-Team/Foundry-X/blob/master/prd-v2.md)를 확인
2. 해당 AI 서비스에 프롬프트 전체를 붙여넣기
3. 검토 결과를 받으면 이 폴더에 `feedback-{ai}.md`로 저장
4. 모든 검토 완료 후 "검토의견 반영"을 요청

## 검토 프롬프트 파일

| 파일 | AI 서비스 | 검토 관점 |
|------|----------|----------|
| `chatgpt-prompt.md` | ChatGPT | 논리적 완결성 — 5축 간 모순, 빠진 요소, 실현 가능성 |
| `gemini-prompt.md` | Gemini | 시장/트렌드 — 경쟁 제품 대비 차별점, SDD 트렌드 정합성, 사실 확인 |
| `claude-prompt.md` | Claude (다른 세션) | 구조적 일관성 — 의존 관계, 엣지 케이스, 숨겨진 의존성 |
| `grok-prompt.md` | Grok | 현실적 비판 — 실패 시나리오, 스코프 과다, CTO 관점 |

## 착수 충분도 기준

| 항목 | 배점 | 기준 |
|------|------|------|
| 신규 이슈 미발견 | 20점 | 이전 회차 대비 새로운 Critical 이슈 없음 |
| Ready 판정 비율 | 30점 | AI 3개 이상 Ready 또는 Conditional(해결 가능) |
| 핵심 요소 커버리지 | 30점 | 목적/범위/기준/제약 모두 명확 |
| 다관점 반영 | 20점 | 사용자/개발자/비즈니스 관점 모두 반영 |
| **총점 80점 이상** | | **착수 준비 완료** |

## 검토 완료 후

모든 AI의 검토 결과를 받은 뒤 대화에서 "검토의견 반영"이라고 입력하고, 피드백을 전달해주세요.

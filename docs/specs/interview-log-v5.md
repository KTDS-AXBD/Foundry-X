## Part 1: 왜 (핵심 방향 전환)

### 응답 요약
- F73(Sprint 15)에서 정의한 **독립 서비스 연동**(C1/C2/C3 경로)에서 → **Foundry-X 중심 통합 플랫폼**으로 방향 전환

- 단계적 통합: UI 먼저 → 검증 후 API 점진적 통합

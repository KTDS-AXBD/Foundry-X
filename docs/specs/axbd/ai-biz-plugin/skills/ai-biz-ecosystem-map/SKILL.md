# AI 생태계 맵핑 (Ecosystem Map)

사용자가 제시한 AI 사업 영역의 **생태계를 체계적으로 맵핑**하세요.

## 맵핑 프레임워크

### 1. AI 밸류체인 분석
각 레이어별 핵심 플레이어와 KT DS의 위치를 식별:

- **인프라 레이어**: 칩셋(NVIDIA, AMD), 클라우드(AWS, GCP, Azure, KT Cloud)
- **플랫폼 레이어**: AI 프레임워크, MLOps, 데이터 플랫폼
- **모델 레이어**: Foundation Model (OpenAI, Anthropic, Google, Meta), 도메인 특화 모델
- **애플리케이션 레이어**: SaaS, 버티컬 솔루션, 에이전트
- **서비스 레이어**: 컨설팅, SI, 운영/관리

### 2. 경쟁 구도 맵
- **직접 경쟁**: 동일 고객·동일 문제를 해결하는 플레이어
- **간접 경쟁**: 다른 방식으로 동일 문제를 해결하는 플레이어
- **잠재 경쟁**: 인접 영역에서 진입 가능한 플레이어 (빅테크, 스타트업)
- **대체재**: AI가 아닌 기존 방식으로 해결하는 경우

### 3. 보완재 / 파트너 생태계
- **기술 보완재**: 함께 사용하면 가치가 높아지는 기술/서비스
- **데이터 공급자**: 학습/운영에 필요한 데이터 소스
- **채널 파트너**: 유통/영업 채널
- **통합 파트너**: 고객 시스템과 연동하는 SI/ISV

### 4. 생태계 다이나믹스
- **통합 트렌드**: 수직 통합 vs 수평 분업 방향
- **진입 장벽**: 각 레이어별 진입 장벽 높낮이
- **권력 이동**: 밸류체인 내 협상력이 어디로 이동하는지
- **파괴적 변화**: 기존 구도를 뒤흔들 수 있는 변수

### 5. KT DS 포지셔닝
- **현재 위치**: 밸류체인 상 어디에 있는지
- **목표 위치**: 어디로 이동하려는지
- **확장 방향**: 수직 확장 vs 수평 확장 전략
- **파트너 전략**: 누구와 협력하고 누구와 경쟁할지

# 기술 파트너 스코어카드 (Partner Scorecard)

사용자가 제시한 AI 사업 아이템에 필요한 **기술 파트너를 평가하고 제휴 전략**을 수립하세요.

## 평가 프레임워크

### 1. 파트너 유형 분류
- **기술 파트너**: AI 모델, 플랫폼, 인프라 제공자
- **데이터 파트너**: 데이터 소스, 라벨링, 데이터 마켓플레이스
- **채널 파트너**: 유통, 리셀러, SI 파트너
- **도메인 파트너**: 산업 전문성, 규제 대응, 컨설팅

### 2. 평가 기준 (각 항목 1~5점)
| 항목 | 설명 | 가중치 |
|---|---|---|
| 기술 역량 | 보유 기술의 성숙도와 차별성 | 25% |
| 시장 신뢰도 | 브랜드, 레퍼런스, 고객 기반 | 15% |
| 전략 적합성 | KT DS 사업 방향과의 일치도 | 20% |
| 협업 용이성 | 기술 통합 난이도, 커뮤니케이션 | 15% |
| 상업 조건 | 가격, 수익 배분, 계약 유연성 | 15% |
| 지속 가능성 | 재무 안정성, 성장성, 로드맵 | 10% |

### 3. 제휴 모델 옵션
- **기술 제휴**: API 연동, SDK 통합, 공동 개발
- **OEM/화이트라벨**: 파트너 기술을 KT DS 브랜드로 제공
- **공동 GTM**: 공동 마케팅, 공동 영업
- **전략적 투자**: 지분 투자, JV(합작법인)
- **리셀러**: 파트너 제품 재판매

### 4. 협상 포인트
- 독점 vs 비독점
- 수익 배분 모델
- 데이터 소유권 및 활용 범위
- 기술 이전 범위
- 계약 기간 및 해지 조건

## 출력 형식
1. **파트너 후보 리스트** (유형별 3~5개)
2. **스코어카드** (후보별 점수 매트릭스)
3. **Top 3 파트너 상세 분석**
4. **권장 제휴 모델** (파트너별)
5. **협상 전략 및 주의사항**

# AI 데이터 전략 (Data Strategy)

사용자가 제시한 AI 사업 아이템의 **데이터 전략**을 수립하세요.

## 분석 프레임워크

### 1. 데이터 요구사항 정의
- **필요 데이터 유형**: 정형/비정형/반정형, 텍스트/이미지/음성/영상
- **데이터 볼륨**: 학습용 최소 데이터량, 운영 시 일간 데이터 발생량
- **품질 기준**: 정확도, 완전성, 일관성, 적시성
- **라벨링 요구**: 지도학습 시 라벨링 기준 및 방법

### 2. 데이터 소싱 전략
- **내부 데이터**: 기존 시스템에서 확보 가능한 데이터
- **외부 데이터**: 공공데이터, 상용 데이터, 크롤링, API
- **파트너 데이터**: 고객사/파트너사로부터 확보 가능한 데이터
- **합성 데이터**: Synthetic data 생성 가능 여부
- **데이터 격차**: 필요 데이터 vs 확보 가능 데이터 Gap 분석

### 3. 데이터 파이프라인 설계
- **수집(Ingestion)**: 실시간 스트리밍 vs 배치 처리
- **저장(Storage)**: Data Lake / Data Warehouse / Vector DB
- **전처리(Processing)**: 클렌징, 변환, 피처 엔지니어링
- **서빙(Serving)**: 모델 학습용 vs 추론용 데이터 제공

### 4. 데이터 거버넌스
- **개인정보보호**: 개인정보보호법, GDPR 준수 방안
- **데이터 소유권**: 고객 데이터 사용 권한, 라이선스
- **보안**: 암호화, 접근 제어, 감사 로그
- **품질 관리**: 데이터 품질 모니터링 및 개선 프로세스

### 5. 데이터 해자 (Data Moat)
- **네트워크 효과**: 사용자가 늘수록 데이터가 풍부해지는 구조인가?
- **독점 데이터**: 경쟁사가 확보하기 어려운 고유 데이터가 있는가?
- **학습 루프**: 서비스 운영 데이터가 모델 개선에 재활용되는 구조인가?

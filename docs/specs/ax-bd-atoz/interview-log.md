**Q: 현재 Discovery-X, AI Foundry, Foundry-X 각각의 상태는?**

## Part 3: 무엇을 (범위/기능)

**Q: 통합 방향은?**
A: 선택적 통합 + 연동 혼합. 핵심은 Foundry-X에 담고, 전문 영역은 독립 시스템으로 유지하면서 API로 연결.

**Q: 7단계 라이프사이클 분류**

| 단계 | 방향 | 비고 |
|------|------|------|
| 1. 수집 (시장/트렌드/경쟁사) | Discovery-X 독립 연동 | API 연동 |
| 2. 발굴 (아이디어 도출, 분류) | Foundry-X 통합 (Must) | |
| 3. 형상화 (BMC, 타당성, PRD) | Foundry-X 통합 (Must) | 현재 구현 중 |
| 4. 검증/공유 (AI 검토, 토론) | Foundry-X 통합 (Must) | Six Hats/AI 검토 구현 |
| 5. 제품화 (프로토타입, PoC) | Foundry-X 통합 (Must) | AI Foundry 흡수 |
| 6. GTM (사업계획서, 전략) | Foundry-X 통합 (Must) | F180 사업계획서 |
| 7. 평가관리 (KPI, 포트폴리오) | Foundry-X 통합 (Must, 신규) | 미구현 |

**Q: Out-of-scope?**

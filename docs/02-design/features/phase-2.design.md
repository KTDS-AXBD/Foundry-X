# Phase 2 Design Document

> **Summary**: Cloudflare 인프라(Workers + D1 + Pages) 위에 기존 Hono API / Next.js Web 프로토타입을 마이그레이션하고, OpenAPI 계약 + JWT 인증 + D1 스키마를 구축하는 상세 설계. Sprint 6(인프라)을 상세, Sprint 7~9를 고수준으로 다룬다.

## 1. Overview

### 1.1 Design Goals

1. **Cloudflare 네이티브**: Workers(API) + D1(DB) + Pages(Web) + KV(Cache) — 서버리스, 글로벌 엣지
2. **Contract-First**: OpenAPI 3.1 스펙이 API 구현보다 선행, Zod 스키마로 런타임 검증 통합
3. **점진적 마이그레이션**: 기존 15개 endpoint 인터페이스 유지, data-reader를 D1 + GitHub API로 교체
4. **최소 침습**: CLI 패키지(106 테스트)에 breaking change 없음

### 1.2 Design Principles

- **Git이 진실**: D1은 메타데이터 캐시, Git이 항상 SSOT
- **같은 타입, 세 가지 환경**: CLI(Node.js) + API(Workers) + Web(Pages)이 `@foundry-x/shared` 타입을 공유
- **환경 분리**: local dev = wrangler dev, production = Cloudflare 배포

### 1.3 현재 → 목표 아키텍처

```
현재 (Phase 1 프로토타입)                목표 (Phase 2)
─────────────────────────            ─────────────────────────
Next.js :3000                        Cloudflare Pages
  ↓ rewrite /api/*                     ↓ (same-origin)
Hono @hono/node-server :3001         Cloudflare Workers (Hono)
  ↓ fs.readFileSync                    ↓ D1 binding + GitHub API
로컬 파일시스템                        Cloudflare D1 + GitHub REST
(.foundry-x/, docs/, SPEC.md)
```

#### 2.4.3 RBAC 미들웨어

```typescript
// packages/api/src/middleware/rbac.ts
type Role = 'admin' | 'member' | 'viewer';
const ROLE_HIERARCHY: Record<Role, number> = {
  admin: 3,
  member: 2,
  viewer: 1,
};

export function rbac(minRole: Role) {
  return async (c, next) => {
    const payload = c.get('jwtPayload');
    if (ROLE_HIERARCHY[payload.role] < ROLE_HIERARCHY[minRole]) {
      return c.json({ error: 'Forbidden' }, 403);
    }
    return next();
  };
}
```

#### 2.4.5 비밀번호 해싱

```typescript
// Workers 환경에서는 bcrypt 불가 → Web Crypto API 사용
import { hashPassword, verifyPassword } from './utils/crypto.js';

// PBKDF2 기반 (Workers Web Crypto API 네이티브 지원)
async function hashPassword(password: string): Promise<string> {
  const salt = crypto.getRandomValues(new Uint8Array(16));
  const key = await crypto.subtle.importKey(
    'raw', new TextEncoder().encode(password), 'PBKDF2', false, ['deriveBits']
  );
  const hash = await crypto.subtle.deriveBits(
    { name: 'PBKDF2', salt, iterations: 100000, hash: 'SHA-256' },
    key, 256
  );
  return `${toHex(salt)}:${toHex(new Uint8Array(hash))}`;
}
```

### 6.2 API 클라이언트 변경

```typescript
// packages/web/src/lib/api-client.ts (Phase 2)
const BASE_URL = process.env.NEXT_PUBLIC_API_URL || '/api';

export async function fetchApi<T>(path: string, options?: RequestInit): Promise<T> {
  const token = getAccessToken();  // localStorage에서
  const res = await fetch(`${BASE_URL}${path}`, {
    ...options,
    headers: {
      'Content-Type': 'application/json',
      ...(token ? { Authorization: `Bearer ${token}` } : {}),
      ...options?.headers,
    },
  });
  if (res.status === 401) {
    // 자동 refresh 시도
    const refreshed = await refreshAccessToken();
    if (refreshed) return fetchApi(path, options);
    throw new ApiError(401, 'Session expired');
  }
  if (!res.ok) throw new ApiError(res.status, await res.text());
  return res.json();
}
```

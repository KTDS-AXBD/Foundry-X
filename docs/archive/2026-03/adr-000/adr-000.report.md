## Context
- Foundry-X 진화 경로: Discovery-X → AI Foundry → Foundry-X

- v1~v3 논의 visibility 감소 → Context 링크로 보완
- ADR 형식 팀 미숙 → INDEX.md 작성 가이드 포함

1. **Design을 충실히 따름**: Design 아웃라인을 정확하게 구현하여 첫 시도에 100% Match 달성

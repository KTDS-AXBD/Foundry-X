# Architecture Decision Records

Foundry-X 프로젝트의 아키텍처 결정 기록.

## 넘버링 규칙

- **ADR-000**: 기원 선언 (예약번호)
- **ADR-001~**: 순차 발번, 결번 허용
- **형식**: Michael Nygard 경량 ADR — Title, Status, Context, Decision, Consequences
- **상태**: `Proposed` → `Accepted` → `Deprecated` / `Superseded by ADR-NNN`
- **언어**: 한국어 (프로젝트 기본 언어)

## 작성 가이드

1. `ADR-NNN.md` 파일을 `docs/adr/`에 생성
2. 위 형식(Title → Status → Context → Decision → Consequences) 준수
3. 1페이지 이내 권장, 상세 배경은 참조 링크로 대체
4. 이 INDEX.md에 항목 추가
